# AI-Based Company-Specific Placement Readiness Analyzer

## Overview
This project analyzes a student's readiness for placements in specific companies based on skills like coding, aptitude, projects, and resume strength.

## Features
- Company-specific readiness score
- Skill gap analysis
- Personalized recommendations
- Dashboard visualization
- Future scope: ML-based prediction

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
- ML (Planned): Python

## Project Structure
- frontend/ → UI
- backend/ → APIs
- ml-model/ → ML logic

## Future Improvements
- ML model integration
- Resume parsing
- Real-time analytics
